Bem-vindo ao meu portfólio!

Sou um desenvolvedor front-end apaixonado pelo que faço. Atualmente, trabalho como freelancer, o que me permite explorar diferentes projetos e desafios. Além disso, estou cursando Análise e Desenvolvimento de Sistemas na Universidade Católica de Brasília, onde tenho a oportunidade de aprimorar minhas habilidades e conhecimentos.

Minhas principais áreas de expertise incluem HTML, CSS e JavaScript, com uma forte ênfase no uso do framework Bootstrap. Com essas ferramentas, sou capaz de criar interfaces atraentes e funcionais, garantindo uma experiência de usuário excepcional.

Principais projetos


O "Perfil de Corretor de Imóveis" é um projeto web desenvolvido utilizando as tecnologias JavaScript, HTML e CSS, que apresenta um perfil fictício de um corretor de imóveis. A página web criada oferece informações detalhadas sobre o corretor, suas habilidades, experiências e realizações, proporcionando aos visitantes uma visão completa do profissional.

A Calculadora de Nota do ENEM é um projeto desenvolvido utilizando as tecnologias web (HTML, CSS e JavaScript) que permite aos usuários calcular sua pontuação final no Exame Nacional do Ensino Médio (ENEM) com base nas notas obtidas nas diferentes áreas de conhecimento.
