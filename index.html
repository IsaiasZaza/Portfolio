Principais projetos


O projeto Pomodoro é uma aplicação web desenvolvida utilizando HTML, CSS e JavaScript, com a integração da biblioteca Bootstrap. O método Pomodoro é uma técnica de gerenciamento de tempo que consiste em ciclos de foco intercalados com pausas. O projeto oferece um temporizador ajustável, permitindo que os usuários definam períodos de trabalho e descanso.

O projeto "Café Entregas" é um website simples e prático que utiliza o framework Bootstrap e JavaScript para criar uma plataforma de pedidos de café online. O foco principal é permitir que os usuários façam pedidos de café de forma conveniente e eficiente.

O "Perfil de Corretor de Imóveis" é um projeto web desenvolvido utilizando as tecnologias JavaScript, HTML e CSS, que apresenta um perfil fictício de um corretor de imóveis. A página web criada oferece informações detalhadas sobre o corretor, suas habilidades, experiências e realizações, proporcionando aos visitantes uma visão completa do profissional.

A Calculadora de Nota do ENEM é um projeto desenvolvido utilizando as tecnologias web (HTML, CSS e JavaScript) que permite aos usuários calcular sua pontuação final no Exame Nacional do Ensino Médio (ENEM) com base nas notas obtidas nas diferentes áreas de conhecimento.
